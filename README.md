# homebrew-tap

Homebrew tap for [jabou](https://github.com/jabou)'s tools.

## Usage

```sh
brew install jabou/tap/sync-exif
```

Or tap first, then install by the short name:

```sh
brew tap jabou/tap
brew install sync-exif
```

## Formulae

- **sync-exif** — apply camera metadata from XMP/XML sidecars to scanned film negatives while preserving the scanner's own metadata as secondary EXIF. [Source](https://github.com/jabou/sync-exif)
